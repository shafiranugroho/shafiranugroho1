Haya | UX Project

03 | User Flows
Challenged by its similarity to sitemaps, this process required visualization and imagination. For this project I created user flows for the perspectives of returning and new users. Overall, the process is a great planning technique that if constructed well, will improve comfort in usability, navigation, wireframing and success in user testing. User flows were further improved by card-sorting activities and surveys. These information captures a sense of user understanding and familiarity that can improve assortment for optimum user flow.
Shafira Nugroho 2017
